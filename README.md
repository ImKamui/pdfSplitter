Разделитель PDF-файлов.
Принцип работы:
1. Указываете путь на один или несколько файлов в формате PDF, которые необходимо разделить на отдельные страницы также в формате PDF.
2. Указываете путь, куда сохранить отдельные страницы. ВАЖНО: Программа создаёт папку, которую называется также, как изначальный PDF-файл, куда уже помещает страницы.
3. Нажимаете на кнопку "Преобразовать".
4. Готово! Файл(ы) был(и) разделен(ы)
